  1. 新建一篇文章
    hexo new <title>

    等同于在 source/_posts/ 下新建一个.md文件

  2. 清除缓存文件(db.json)和已生成的静态文件(public)
    hexo clean
  3. 生成静态文件
    hexo g
  4. 启动服务器。默认情况下，访问网址为：http://localhost:4000/
    hexo s
